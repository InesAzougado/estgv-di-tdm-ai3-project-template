Início

Benvindo ao site do nosso projeto prático de Aplicações para a Internet III.

Este é um projeto que consiste em:

▸ Desenvolver uma API em NodeJS (ou PHP) para disponibilizar serviços a uma aplicação cliente;

▸ Desenvolver uma aplicação cliente através da framework Angular (ou vanilla JS + JQuery) para interagir com a API anterior;

▸ Integrar a utilização de serviços (APIs) de terceiros na API para implementar algumas das funcionalidades a fornecer ao utilizador;
